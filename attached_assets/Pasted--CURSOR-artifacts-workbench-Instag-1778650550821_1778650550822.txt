Нормализация ответа:

function pickFirstString(...values: unknown[]) {
  for (const value of values) {
    if (typeof value === "string" && value.trim()) return value;
  }
  return "";
}

function collectImageUrls(item: Record<string, unknown>) {
  const urls: string[] = [];

  const pushUrl = (value: unknown) => {
    if (typeof value === "string" && /^https?:\/\//.test(value)) {
      urls.push(value);
    }
  };

  pushUrl(item.displayUrl);
  pushUrl(item.imageUrl);

  if (Array.isArray(item.images)) {
    item.images.forEach(pushUrl);
  }

  const childPosts = item.childPosts;
  if (Array.isArray(childPosts)) {
    childPosts.forEach((child) => {
      if (child && typeof child === "object") {
        const c = child as Record<string, unknown>;
        pushUrl(c.displayUrl);
        pushUrl(c.imageUrl);
      }
    });
  }

  const carouselMedia = item.carouselMedia;
  if (Array.isArray(carouselMedia)) {
    carouselMedia.forEach((media) => {
      if (media && typeof media === "object") {
        const m = media as Record<string, unknown>;
        pushUrl(m.displayUrl);
        pushUrl(m.imageUrl);
      }
    });
  }

  return Array.from(new Set(urls));
}

  const blob = new Blob([buffer], { type: mimeType });

async function remoteImageToFalUrl(imageUrl: string) {
  const r = await fetch(imageUrl);
  if (!r.ok) throw new Error(`Failed to fetch image for fal: ${r.status}`);
  const mimeType = r.headers.get("content-type") || "image/jpeg";

16. ЖЁСТКАЯ ПРОВЕРКА
Проверка 1. Старое не сломано

Открыть рабочее пространство.

Должно быть 3 плитки:

Telegram Tracker
Threads — пост + картинка
Instagram Carousel Remix

Проверить:

Telegram Tracker открывается/работает как раньше.
Threads — пост + картинка открывается как раньше.
Новая плитка открывает отдельную модалку.
Проверка 2. Import

Вставить Instagram URL.

Нажать:

Import from Instagram

Ожидаемый результат:

- caption заполнился
- появились preview слайдов
- количество preview равно количеству imported slides
Проверка 3. Analyze

Нажать:

Analyze carousel

Ожидаемый результат:

- появилась карточка на каждый слайд
- есть Original OCR text
- есть Visual description
- badges отображаются только при true
Проверка 4. Rewrite

Нажать:

Rewrite carousel text

Ожидаемый результат:

- у каждого слайда появился rewrittenText
- rewrittenText можно редактировать руками
- количество rewritten slides не изменилось
Проверка 5. Generate

Загрузить user photo.

Нажать:

Generate carousel images

Ожидаемый результат:

- у каждого слайда появился generatedImageUrl
- изображения отображаются в UI
- если hasFace=false, userPhoto не должен добавляться как reference
- если hasFace=true, userPhoto должен добавляться как второй reference image
Проверка 6. Ошибки

Проверить:

Без APIFY_TOKEN import возвращает 503.
Без APIFY_ACTOR_ID import возвращает 503.
Без OPENROUTER_API_KEY analyze/rewrite возвращает 503.
Без FAL_KEY generate возвращает 503.
Пустой Instagram URL возвращает 400.
Не Instagram URL возвращает 400.
Без importedSlides и без manual upload Analyze не запускается.
