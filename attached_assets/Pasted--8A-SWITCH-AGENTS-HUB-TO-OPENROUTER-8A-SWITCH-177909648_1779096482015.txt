ШАГ 4 — УДАЛИТЬ extractAnthropicText:
Найти функцию:

function extractAnthropicText(response: Anthropic.Message): string {
  ...
}

Удалить полностью.

На её место добавить helper:

function extractOpenRouterText(completion: Awaited<ReturnType<typeof openrouter.chat.completions.create>>): string {
  const content = completion.choices[0]?.message?.content;

  if (typeof content === "string") {
    return content.trim();
  }

  if (Array.isArray(content)) {
    return content
      .map((part) => {
        if (part && typeof part === "object" && "text" in part) {
          const text = (part as { text?: unknown }).text;
          return typeof text === "string" ? text : "";
        }
        return "";
      })
      .filter(Boolean)
      .join("\n\n")
      .trim();
  }

  return "";
}

Если TypeScript ругается на сложный ReturnType, сделать проще:

function extractOpenRouterText(completion: {
  choices?: { message?: { content?: unknown } }[];
}): string {
  const content = completion.choices?.[0]?.message?.content;

  if (typeof content === "string") {
    return content.trim();
  }

  if (Array.isArray(content)) {
    return content
      .map((part) => {
        if (part && typeof part === "object" && "text" in part) {
          const text = (part as { text?: unknown }).text;
          return typeof text === "string" ? text : "";
        }
        return "";
      })
      .filter(Boolean)
      .join("\n\n")
      .trim();
  }

  return "";
}

Выбрать вариант, который проходит typecheck без изменения других файлов.

ШАГ 5 — ПЕРЕПИСАТЬ POST /wb/agents/:key/messages:
В route:

app.post("/wb/agents/:key/messages", async (req, res) => { ... })

Найти блок:

const client = getAnthropicClient();
let response: Anthropic.Message;
try {
  response = await client.messages.create({
    model,
    max_tokens: 4096,
    system,
    messages: [
      ...history.map((message) => ({
        role: message.role,
        content: message.content,
      })),
      { role: "user", content },
    ],
  });
} catch (e) {
  ...
}

const assistantText = extractAnthropicText(response);

Заменить на OpenRouter:

let completion: Awaited<ReturnType<typeof openrouter.chat.completions.create>>;
try {
  completion = await openrouter.chat.completions.create({
    model,
    max_tokens: 4096,
    messages: [
      { role: "system", content: system },
      ...history.map((message) => ({
        role: message.role,
        content: message.content,
      })),
      { role: "user", content },
    ],
  });
} catch (e) {
  const detail = e instanceof Error ? e.message : String(e);
  return res.status(502).json({
    error: "OpenRouter request failed",
    detail,
  });
}

const assistantText = extractOpenRouterText(completion);
if (!assistantText) {
  return res.status(502).json({
    error: "OpenRouter request failed",
    detail: "Empty assistant response",
  });
}

КАКИЕ ENV ТЕБЕ НУЖНЫ БУДУТЬ ПОСЛЕ ПРАВКИ

В artifacts/workbench/.env:

OPENROUTER_API_KEY=твой_ключ_openrouter

OPENROUTER_AGENTS_MODEL_EXECUTOR=anthropic/claude-sonnet-4

Если anthropic/claude-sonnet-4 в OpenRouter недоступен или дорогой, временно можно поставить любую доступную модель, например:

OPENROUTER_AGENTS_MODEL_CEO=google/gemini-2.5-pro
OPENROUTER_AGENTS_MODEL_EXECUTOR=google/gemini-2.5-flash

или другую модель, которая реально есть у тебя в OpenRouter.

Я не фиксирую конкретную модель как обязательную, потому что доступность и цены моделей в OpenRouter могут меняться. Модель должна жить только в .env.
